# 🚀 Projeto Informativo – Semana das Engenharias

Site informativo desenvolvido para apresentar um projeto acadêmico do curso de Engenharia da Computação, exposto na Feira de Projetos da Semana das Engenharias da Universidade São Francisco.

---

## Objetivo

Criar uma página informativa para divulgar e explicar o funcionamento do projeto desenvolvido em equipe, tornando o conteúdo acessível e organizado para visitantes da feira.

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- Responsividade básica
- Design de interface

---

## Estrutura do Projeto

- index.html  
- Pasta assets (imagens do projeto)  
- Pasta css (estilização)
